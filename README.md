<h1 align="center">Rocket Launcher</h1>




---

**Lancer l'application**

```console
> npm start
```
---

**Création du build**

| Platform    | Command              |
| ----------- | -------------------- |
| Windows  | `npm run build:win`   |
| macOS    | `npm run build:mac`   |
| Linux    | `npm run build:linux` |
| All    | `npm run build:all` |
